# 🚀 Quickstart

Learn by deploying a real DeFi lending system. From simple ERC20 tokens to complex Diamond Proxy architectures.

## 📦 Installation

Install the Fever CLI globally using npm:

```bash
npm install -g @fevertokens/cli
```

Verify the installation:

```bash
fever --version
```

You should see the version number displayed (e.g., `v2.0.0`).

## 🎯 What You'll Learn

In this quickstart guide, you'll:

- 🔐 Authenticate with the Fever platform
- 📁 Create and manage projects
- 🔨 Compile Solidity smart contracts
- 🚀 Deploy a simple ERC20 token
- 💎 Deploy a complex multi-contract Diamond Proxy system
- 🔍 Track your deployments on the web platform
